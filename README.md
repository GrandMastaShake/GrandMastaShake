### Main projects

**[roblox-modular-lib](https://github.com/GrandMastaShake/roblox-modular-lib)**  
49 `--!strict` Luau modules for building live Roblox games — pets, trading, inventory, XP, quests, leaderboards, DataStore save/load, and more. Full dependency injection, EventBus architecture, 54 test files. Built as the toolkit under an Adopt Me-style pet game demo.

**[bits-and-baubles-blender-kit](https://github.com/GrandMastaShake/bits-and-baubles-blender-kit)**  
Python/Blender toolkit for procedural brick-style asset building. Covers the full pipeline: mesh generation → rig → PBR bake → Roblox-ready FBX export. Asset validation built in so nothing fails the upload.
